## About the Project

This is a **Weather App** built using **HTML, CSS, and JavaScript** that fetches live weather data from a **public API** using the **Fetch API**. Users can search for a city and instantly view its current weather details.

This project is developed as part of **Task 10: Consume a Public API Using Fetch API**.

---

## Features

* Search weather by city name
* Uses a **free public weather API** (no authentication required)
* Fetches live data using JavaScript `fetch()`
* Displays weather information dynamically:

  * City name
  * Temperature
  * Weather conditions
* Loading indicator while data is being fetched
* Error handling for invalid city names and network issues
* Clean and simple user interface

## Technologies Used

* HTML
* CSS
* JavaScript (Fetch API)
